## [0.1.0] - 2026-05-18

### Added

- Initial release of the DevTools extension UI.
- Two-column inspector with live updates, request list and detail view.
- Method, status and search filters.
- Pretty JSON viewer with copy-to-clipboard.
- Copy as cURL action.
- Error highlight, duration and timestamp display.
- Dark mode inherited from the DevTools host theme.
